### 学习记录
1. TDD 流程有效：先写测试，再实现，最后重构
2. Ponder Gate 触发逻辑：step=0 时无条件执行，后续根据不确定性判断
3. 自适应配置：序列长度 <128 用 base_steps，>1024 用 max_steps，中间插值
4. 梯度自适应：大梯度 -> 小 LR，小梯度 -> 大 LR（反比关系）
