# 解释文档

欢迎来到 ADN 解释文档！这里帮助你理解 ADN 的设计决策、架构演进和研究背景。

---

## 💡 文档目的

与教程和指南不同，解释文档：
- 回答 **"为什么"** 而不是 **"怎么做"
- 提供背景知识和理论依据
- 帮助读者形成系统性理解

## 📝 写作风格

解释文档采用以下风格:

- **叙述性**: 像讲故事一样解释概念
- **比较性**: 对比不同选择的优劣
- **历史视角**: 说明决策的演变过程
- **引用丰富**: 引用论文和相关资源

## 🔗 与其他文档的关系

```
解释文档 (理解导向)
    ↑
教程 (学习导向) ──> 操作指南 (任务导向)
    ↓
参考文档 (信息导向)
```

- **教程** 回答 "如何学习"
