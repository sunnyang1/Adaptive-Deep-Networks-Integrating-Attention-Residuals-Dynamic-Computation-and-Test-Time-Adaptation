## 📋 前置知识

学习 ADN 之前，建议具备：

- **Python**: 熟悉 Python 3.10+ 语法
- **PyTorch**: 了解张量操作和模型训练基础
- **Transformer**: 理解注意力机制基本概念
- **命令行**: 熟悉基本的 shell 命令

如果缺少某些知识，可以先查看：
- [PyTorch 官方教程](https://pytorch.org/tutorials/)
- [The Illustrated Transformer](https://jalammar.github.io/illustrated-transformer/)
