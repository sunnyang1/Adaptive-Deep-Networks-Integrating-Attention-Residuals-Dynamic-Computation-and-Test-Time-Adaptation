# 产品简报: 增量 KV Cache 新模型结构

## 战略愿景
实现真正的 O(L) 增量 KV Cache 更新，解决长序列生成时的计算瓶颈，使 256K 上下文推理变得实用。

## 目标用户
- 研究人员: 长序列实验 (>32K)
- 部署工程师: 生产环境高吞吐推理

## 成功指标
1. 长序列 (>4K) 推理速度提升 5-10x
2. 内存使用保持稳定（不随序列长度线性增长）
3. 数值输出与原始实现一致

## 约束条件
- 数值精度: 与原始实现差异 < 1e-6
- API 兼容: 保留现有 generate() 行为
- 设备支持: CPU/MPS/CUDA

## 关键决策
1. **包装器模式**: IncrementalGenerator 管理状态，不修改模型
2. **显式状态**: 所有中间状态显式跟踪，便于调试
3. **懒加载**: 只在需要时初始化增量结构
