# Auto-Research Skill 创建记录

## 2026-03-26

### 创建了「自动写论文」Skill
- **位置**: `~/.workbuddy/skills/自动写论文/`
- **架构**: 参考 [AutoResearchClaw](https://github.com/aiming-lab/AutoResearchClaw) 的 8 阶段研究流水线
- **实现方式**: 纯 Prompt + Python 脚本（无需安装 AutoResearchClaw）
- **流程**: 选题 → 文献检索 → 知识合成 → 实验设计 → 实验执行 → 分析决策 → 论文撰写 → 最终化导出
- **文献源**: ArXiv API（复用已有 ArXiv 论文追踪 skill）
- **引用**: CrossRef API + 学术引用管理 skill
- **输出格式**: Markdown + LaTeX + Word + BibTeX
- **特性**: 断点续跑、人工审核门（Phase 3/6）、实验沙盒、防伪造机制
- **安全审计**: 通过朱雀实验室安全扫描，无恶意行为
- **测试**: 所有模块导入测试 + 端到端功能测试通过
